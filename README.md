# PokedexCLI
A command line interface which will serve you as a asistant in your journey in the mystirious pokemon world

# How to use
## Install [go](https://go.dev)

## Clone this repo
```bash
git clone https://github.com/nabin3/pokedexcli.git
```
## Build the app
```bash
cd pokedexcli
go build
```
## Run the app
```bash
./pokedexcli
```

Thats it
